# Isaac web runtime assets

This directory intentionally contains only assets loaded by the generated website.
Large AI source images, intermediate composites, unused icon crops, copied reference
textures, and generation prompts are excluded to keep the deployed repository small.

| Runtime asset | Use |
|---|---|
| `basement-room.webp` | Homepage and compressor Basement background |
| `basement-panel.webp` | Scalable Basement menu/card borders |
| `challenge-page-background.webp` | Challenge-page paper surface with separated Hero Menu doodles |
| `challenge-sheet.webp` | Scalable central challenge sheet |
| `paper-surface.webp` | Challenge-page fallback paper texture |
| `repentance-logo.png` | Homepage and compressor logo |
| `menu-cursor.png` | Isaac menu cursor |
| `inconsolata-bold.ttf` | Code and HUD font |
| `isaac-pixel-subset.woff2` | Subsetted display font used by the current Chinese UI |

The visual assets are derived from the user's local installation of The Binding of
Isaac: Rebirth / Repentance+ and from reference-guided `gpt-image-2` edits made from
those resources. The Binding of Isaac and its artwork belong to their respective
rights holders.
